# Rusty-Passwords

A code to be able to add passwords and ask people to log in using the passwords. Built in Rust.

## Instructions for Build and Use

Steps to build and/or run the software:

1. clone repository
2. make sure rust is installed https://rust-lang.org/tools/install/
3. cd to the folder and run rustc main.rs
4. run .\main

Instructions for using the software:

1. first put in a psswored admin is a default password to us
2. After that you can add or removoe passwords
3. Then you can run whenever you like and use passwords that you have added to the list
